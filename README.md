## 🎯 Visão Geral

A **Robycorp** é uma empresa especializada em **Tecnologia da Informação**, oferecendo soluções digitais inovadoras e eficientes para impulsionar negócios. Este site institucional foi desenvolvido com **100% frontend**, design moderno dark theme, animações sofisticadas e experiência de usuário premium.

### ✨ Destaques
- 🎨 Design moderno e elegante com tema dark
- 📱 100% responsivo (desktop, tablet, mobile)
- ⚡ Sem dependências de backend
- 🔊 Efeitos sonoros interativos
- ✝️ Elementos espirituais sutis e harmoniosos
- 📊 Google Forms integrado para contatos
- 🎭 Animações suaves e profissionais

## 📦 Instalação e Uso

### Requisitos
- Navegador moderno
- Sem backend
- Sem compilação necessária

### Como abrir
1. Baixe ou clone o projeto
2. Abra `index.html` no navegador

### Servidor local (opcional)
```bash
# Python 3
python -m http.server 8000

# Node.js
npx http-server
```
